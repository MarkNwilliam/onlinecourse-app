# Online Course App

A Django-based online course platform with assessment features including mock exams, questions, and grading.

## Features

- Course and lesson management
- Mock exams with multiple-choice questions
- Automated grading and result feedback
- Admin interface for content management
- Bootstrap-based responsive UI

## Tech Stack

- Python / Django
- SQLite
- Bootstrap 5
- GitHub Actions (CI/CD)
- Tekton Pipelines (OpenShift)
